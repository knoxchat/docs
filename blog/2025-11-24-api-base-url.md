# KnoxChat Updates: API Base URL

### Before:
- OpenAI API Compatible Base URL: `https://knox.chat/v1`
- Standard API Base URL: `https://knox.chat`

### After/Now
- OpenAI API Compatible Base URL: `https://api.knox.chat/v1`
- Standard API Base URL: `https://api.knox.chat`
